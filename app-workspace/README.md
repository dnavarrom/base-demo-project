# App Workspace

Este es el directorio de trabajo donde reside el código de la aplicación. 

## Stack Técnico

El proyecto está configurado con el siguiente stack tecnológico obligatorio:

- **Lenguaje**: TypeScript estricto.
- **Backend**: Node.js.
- **Despliegue y Testing Local**: Docker y Docker Compose (Pendiente configuración de contenedores).
- **Gestor de Paquetes**: `npm`.

## Estructura Inicial

Se ha inicializado un proyecto base de Node.js con la configuración de TypeScript. Aún no se ha desarrollado ninguna lógica de negocio y estamos a la espera de la fase de planificación de requerimientos y diseño de arquitectura funcional.
